Course description:

Train AI agents to crush games. (no boring theory) We take SpaceInvaders as a starting ground & go through everything needed for the Reinforcement Learning (RL) pipeline.👾

✅ Carefully structured ~9h course



🦾 We use Godot, the most popular free game engine — intuitive & if you already know Python it's a perfect fit



🧩 Modular AI code — easily copied for other games or your own crazy 2D/3D envs



💡 Practice RL without a dead end setup — Godot is versatile & you can use any RL algorithm with it



🚫 No ChatGPT — we cover the deep RL algorithms PPO & BTR



🧪 Methodical training with WandB (a popular data science tool you will learn about)



👁️ The agent sees the game via screenshots — almost like a human would



🚀 No fancy PC needed – BTR crushes games on potato hardware
